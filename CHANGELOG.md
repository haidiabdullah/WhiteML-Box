# Changelog

## v0.7.2

- Improved GUI sizing and startup behaviour on Windows.
- Added vertical scrolling to Data, Preprocess, Model, and Mapping panels.
- Fixed map-preview colour-range handling so preview stretching does not alter exported GeoTIFF values.
- Improved launcher behaviour for Windows use.
- Included SoftwareX-ready documentation and example data package.

## v0.7.0

- Added optional sensor band-centre conversion.
- Added target sensor presets for Landsat, Sentinel-2, PRISMA, DESIS, EnMAP, and CHIME.
- Added mapping support for models trained with sensor conversion.
- Added Help/workflow tab and WhiteML-Box logo.

## v0.6.0

- Renamed the tool to WhiteML-Box.
- Added all-pair spectral indices.
- Added continuous wavelet features.
- Improved feature-name export for generated indices and wavelet features.

## v0.5.1

- Added raster-based GeoTIFF prediction mapping.
- Added mapping-ready model storage and mapping configuration sidecar export.
